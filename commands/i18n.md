---
description: Extract i18n strings and check translation completeness
argument-hint: [locale-or-scope]
allowed-tools: Read, Grep, Glob, Bash, mcp__codegraph__*, mcp__code-review-graph__*
---

Always execute via the Workflow engine.

```
∴ coder-orchestrator [T1] → Workflow(i18n-audit): Extract i18n strings + check translation coverage

∴ Workflow({
  name: 'i18n-audit',
  description: 'Extract untranslated strings, check coverage, flag missing translations',
  phases: [
    { title: 'Extract',  detail: 'parallel: source strings + existing translations' },
    { title: 'Diff',     detail: 'find missing keys, stale keys, untranslated strings' },
    { title: 'Report',   detail: 'coverage matrix + action items' },
  ],
})

phase('Extract')
const [sourceStrings, translationFiles] = await parallel([
  () => agent(
    `Run mcp__codegraph__extract_i18n_strings on scope: ${$ARGUMENTS || 'full project'}.
    Also grep for: t(), i18n.t(), $t(), useTranslation patterns.
    Return: all string keys found in source code.`,
    { label: 'source-strings', phase: 'Extract', agent: 'coder-workflow:explore-codebase' }
  ),
  () => agent(
    `Read all translation files (locales/*.json, i18n/*.ts, messages/**).
    Return: all available locales + their key counts.`,
    { label: 'translation-files', phase: 'Extract', agent: 'coder-workflow:explore-codebase' }
  ),
])

phase('Diff')
const diff = await agent(
  `Compute i18n coverage diff:
  - Missing keys: in source but not in translation files
  - Stale keys: in translation files but not in source
  - Coverage per locale: X/Y keys translated
  Source strings: ${sourceStrings}
  Translations: ${translationFiles}`,
  { label: 'i18n-diff', phase: 'Diff', agent: 'coder-workflow:explore-codebase' }
)

phase('Report')
const report = await agent(
  `Produce i18n coverage report:
  1. Coverage matrix: locale vs % complete
  2. Missing keys list (must add)
  3. Stale keys list (safe to remove)
  4. Untranslated source strings (needs t() wrapper)
  Diff: ${diff}`,
  { label: 'i18n-report', phase: 'Report' }
)

return { report }
```
